# Milestones — Design-to-Code Playground

## M1 — Playground Shell
- Tri-panel UI (Code, Preview, Reference).
- Basic static example project wired to preview.
- Local dev environment documented.

## M2 — Figma Import
- Figma OAuth/token support.
- Node tree → intermediate representation.
- Single-screen TSX generation (no UI library).

## M3 — URL Import
- DOM + CSS fetcher.
- DOM → intermediate representation.
- Single-screen TSX/HTML from URL.

## M4 — UI Library Adapters
- ShadCN + Tailwind mapping.
- MUI + custom theme generator.
- AntD mapping with default theme.

## M5 — Tooling Integrations
- VS Code extension (commands + Webview).
- Claude Code hooks for natural language workflows.
- Example templates and documentation.

## M6 — Release Hardening
- Test suite on reference designs and URLs.
- Performance and fidelity benchmarks.
- Public docs, versioned release (v1.0.0).
