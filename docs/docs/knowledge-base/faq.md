# Frequently Asked Questions

## General

### How do I search the documentation?
Use the search bar at the top of the page or press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac) to open quick search.

### How do I suggest changes to documentation?
Click "Edit this page" at the bottom of any documentation page to submit changes via GitHub pull request.

### Who maintains this documentation?
The Operations Excellence team maintains the documentation platform, while individual departments are responsible for their content.
