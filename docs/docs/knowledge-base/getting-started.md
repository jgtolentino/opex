# Getting Started with OpEx Docs

## For New Employees

Welcome! This guide will help you navigate the OpEx documentation system.

### First Steps

1. **Read the Introduction**: Start with the [introduction](/docs/knowledge-base/introduction) to understand the documentation structure
2. **Review Your Department Policies**: Navigate to your department's section (HR, Finance, Operations, etc.)
3. **Bookmark Important Pages**: Save frequently-accessed pages for quick reference
4. **Familiarize Yourself with Workflows**: Review common workflows relevant to your role

### Navigation Tips

- **Use the Sidebar**: Browse by category using the left sidebar
- **Search**: Use Ctrl+K (or Cmd+K on Mac) to open search
- **Mobile Friendly**: Access documentation from any device
