## How to Use This Documentation

### Navigation
Use the sidebar on the left to browse documentation by category. Each major section (HR, Finance, Operations, Technical) has its own dedicated sidebar for easy navigation.

### Search
Use the search bar at the top to quickly find specific topics, policies, or procedures.

### Contribute
This documentation is version-controlled in Git. To suggest changes or improvements:
1. Visit the [GitHub repository](https://github.com/jgtolentino/opex)
2. Click "Edit this page" at the bottom of any documentation page
3. Submit a pull request with your changes
