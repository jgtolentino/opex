## How to Use These Templates

### General Instructions

1. **Download** the appropriate template for your need
2. **Complete** all required fields (marked with asterisks)
3. **Attach** supporting documentation as noted
4. **Submit** via appropriate workflow (ServiceNow, email, etc.)
5. **Track** approval status in system

---

### Template Customization

**Allowed**:
- Add additional rows for line items
- Insert company logos (approved versions only)
- Expand justification text boxes

**Not Allowed**:
- Remove required fields
- Alter formulas or calculations
- Change approval routing
- Modify version numbers
