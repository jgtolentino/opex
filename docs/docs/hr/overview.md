# Human Resources Documentation

Welcome to the HR documentation hub. This section contains all HR-related policies, workflows, templates, and resources.

Comprehensive HR policies covering:

- Remote work guidelines
- Code of conduct and ethics
- Employee benefits
- Performance management

### Workflows
Step-by-step process documentation with BPMN diagrams:
- **[Employee Requisition](/docs/hr/workflows/employee-requisition)** - Complete hiring request and approval workflow

Ready-to-use forms and documents:
